# 🚀 ET Sentinel – AI-Powered Tax Optimization System

ET Sentinel is an AI-driven financial intelligence system that analyzes Form 16 documents and provides actionable tax optimization insights.  
It transforms raw financial data into advisor-level recommendations.

---

## 🔥 Features

- 📄 **Form 16 Analysis**
  - Extracts salary, tax, and deductions automatically
- 🧠 **AI Financial Advisor**
  - Provides intelligent tax-saving recommendations
- 📊 **Wealth Projection**
  - 20-year growth simulation using CAGR
- 🏦 **Tax Optimization Engine**
  - Detects unused deductions (80C, NPS, etc.)
- 👨‍👩‍👧 **Household Insights**
  - Financial alignment and health scoring
